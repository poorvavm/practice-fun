Behavioral Patterns
=============================
	The Behavioral Pattern is the group of eleven design patterns, namely,
	-  Chain of Responsibility
	-  Command Pattern
	-  Interpreter Pattern
	-  Iterator Pattern
	-  Mediator Pattern
	-  Memento Pattern
	-  Observer Pattern
	-  State Patterns
	-  Strategy Patterns
	-  Template Method Patterns
	-  Visitor Patterns
